# bj-telecom-iptv
Beijing Telecom IPTV m3u playlist


# 节目源文件说明： 

- 只需要下载[bj-telecom-iptv.m3u](https://github.com/wuwentao/bj-telecom-iptv/blob/master/bj_telecom_iptv.m3u)文件即可使用，播放器打开该文件既可自动导入所有频道信息

或提交merge request,共同维护与更新准确的节目源列表
